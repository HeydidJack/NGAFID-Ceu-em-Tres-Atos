ConvTok-MWLA (Multi-Window Local Attention). Extending the sliding window concept, MWLA equips each attention head with an independent, learnable window half-width wi, enabling concurrent multi-scale local feature extraction within a single layer. Rather than using a uniform window size across all heads, MWLA simultaneously employs narrow windows for fine-grained transient detection and wider windows for medium-range pattern aggregation, effectively capturing hierarchical temporal structures without increasing network depth or computational width. This multi-resolution approach proves particularly effective for Fault Classification on NGAFID's long-tail fault distribution, where different fault types may manifest across varying temporal scales (e.g., sudden sensor spikes vs. gradual thermal drift), achieving the highest F1 scores (51.76%) among all tested architectures after balanced augmentation.
