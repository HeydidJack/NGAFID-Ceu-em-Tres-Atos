CNN (Shallow One-Dimensional Convolutional Network). This baseline implements a lightweight convolutional architecture for local temporal feature extraction. The input tensor with dimensions (Batch, Length, Dimension) is first transposed to (Batch, Dimension, Length) to conform to PyTorch Conv1d conventions. A single convolutional layer with kernel size 3, stride 1, and padding 1 transforms the input channels to hidden_dim feature maps, preserving sequence length. This is followed by ReLU activation and MaxPool1d with kernel size 2 and stride 2, which halves the temporal resolution. The pooled features are flattened and fed into a linear classifier. While computationally efficient, this architecture possesses a limited receptive field and shallow depth, making it suitable primarily for detecting simple local patterns rather than modeling complex long-range temporal dependencies inherent in flight data.
