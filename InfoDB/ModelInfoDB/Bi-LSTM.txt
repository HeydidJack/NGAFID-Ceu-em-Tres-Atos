Bi-LSTM (Bidirectional Long Short-Term Memory with Channel Projection). This recurrent architecture addresses sequential temporal dependencies through a deep bidirectional design. To reduce computational complexity, the model first applies a 1x1 convolutional projection that reduces the input feature dimensionality to 64 channels. The projected sequence then feeds into a 3-layer bidirectional LSTM (num_layers=3) with hidden dimension d_model, processing data in both forward and backward temporal directions to capture past and future context simultaneously. The architecture extracts the final hidden state from the last time step, concatenating the forward and backward representations to yield a vector of size 2 times d_model. This passes through a regularized projection head comprising LayerNorm, Dropout with rate 0.3, a Linear layer mapping to 128 dimensions, ReLU activation, another Dropout(0.3), and the final classification layer. This design effectively captures long-range global temporal patterns, making it particularly suitable for the Long-stage anomaly detection task where understanding the entire flight sequence context is critical.
