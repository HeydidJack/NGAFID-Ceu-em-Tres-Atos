MLP (Multi-Layer Perceptron). This model serves as the most fundamental baseline, treating multivariate time series classification as a static pattern recognition task without exploiting temporal ordering. The architecture begins with a flattening operation that collapses the input tensor of shape (Batch, Sequence Length, Feature Dimension) into a single vector. This flattened representation passes through a two-layer fully connected network with ReLU nonlinearity, where the first linear layer projects from the input dimension to a hidden dimension (d_model), and the second layer maps to the target number of classes (clasnum). Due to its complete ignorance of temporal structure and sequential dependencies, this model establishes a lower-bound performance baseline, useful for verifying whether temporal modeling provides significant advantage over simple feature vector classification in aircraft health monitoring scenarios.
